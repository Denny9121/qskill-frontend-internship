# qskill-frontend-internship
QSkill Frontend Internship Project

This project is developed as part of the QSkill Frontend Internship Assignment using React, Vite, React Router, and Tailwind CSS.

Features
🌐 English Text Translator
🔑 Random String Generator
⚛️ React Hooks (useState, useEffect, useCallback)
🧭 React Router for navigation
📱 Responsive and modern UI
🎨 Tailwind CSS styling
⚡ Fast development with Vite
🧩 Reusable React components


Technologies Used
React.js
Vite
JavaScript (ES6+)
React Router DOM
Tailwind CSS
HTML5
CSS3
